# Java
* Java is Programming Language which is 99 % Object Oriented Programming Language
* It follows WORA (Write Once Run EveryWhere)

## JRE
 Java Runtime Environment is the runtime environment in which libraries , jvm which provides runtime for code to run
## JVM
Java virtual machine is the one which executes byte code into machine instructions
## JDK
Java Development Kit which  contains compiler , JRE , and other tools required for development

## Java Code
Human Readable and has .java extension
## Byte Code
Generated by compiler and has .class extension

## How Java Code runs?
1. Java code written and understood by human (.java)
2. Compiler compiles and checks syntax errors and converts into byte code (.class)
3. ByteCode is platform independent and can run on any machine jvm installed on it
4. Jvm executes bytecode into machine instructions

## JVM is platform Dependent
Jvm is Platform dependent as it depends on machine instructions in which it executes

## Java ByteCode is Platform Independent
Java ByteCode can run on any  machine that has jvm installed on it 


## How to Download java in windows

1. download jdk from chrome
2. Copy path up to bin from directory stored
3. go to environment variables and add PATH variable
4. open command prompt

```cmd
java --version
```
