# Variables and Data Types

1. Variables are the labels to the values
2. Data Types define the what kind of data it stores and what kind of operation it do

# Primitive Data Types
1. Primitive Data Types are data types which stores values
2. Primitives are faster.

## Integer
It stores numerical types (default : 0)

1. byte : 8bits (-128 to 127)
2. short : 16bits (-2<sup>15</sup> to 2<sup>15</sup> - 1)
3. int : 32 bits (-2<sup>31</sup> to 2<sup>31</sup> - 1)
4. long : 64 bits (-2<sup>63</sup> to 2<sup>63</sup> - 1) : suffix : L or l (mandatory)

## Decimal Numbers
It stores floating point numbers(default : 0.0)

1. float : 32bits (atmost 7 precisions) : suffix f or F (mandatory)
2. double : 64bits (atmost 15 precisions)  : suffix d or D (optional)

## char (default :\u0000)
char data type : data type stores character value 

## boolean (default : false)
boolean represents true or false 

# Reference Data Types
Reference Data Types are data types which stores references and references value to heap

## Arrays 
Arrays is Data Structure which stores the group of data types

## Strings
Strings is the data type which stores characters
